## Как обновить GitHub Pages

1. Распакуй архив.
2. В репозитории GitHub открой Add file → Upload files.
3. Загрузи файлы из папки `friday_v0_1_1_pwa` в корень репозитория с заменой старых:
   - index.html
   - style.css
   - app.js
   - manifest.json
   - service-worker.js
   - README.md
   - icons/
4. Commit changes.
5. Подожди 1–3 минуты.
6. На iPhone открой сайт и обнови страницу.

Если старая версия не обновилась, открой сайт в Safari, обнови страницу 2–3 раза или удали старую иконку с рабочего стола и добавь заново.
