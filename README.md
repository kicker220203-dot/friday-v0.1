# Пятница v0.2.3-dev PWA

Dev-версия без обычного service worker cache.

## Зачем эта версия

На iPhone открывалась старая версия Пятницы из кэша. Эта сборка сделана для разработки: она чистит старые service worker caches и не регистрирует новый постоянный кэш.

## Что должно быть видно

- В меню: `Пятница v0.2.3-dev`.
- На стартовом экране: кнопка `Включить голос`.
- В меню есть `Тест голоса`.
- Голосовые ответы сохранены.

## Как обновить GitHub Pages

1. Распакуй архив.
2. Загрузи содержимое папки `friday_v0_2_3_dev_pwa` в корень репозитория с заменой старых файлов.
3. В Commit changes напиши:
   `Update to v0.2.3-dev — disable cache`
4. Подожди 1–3 минуты.
5. На iPhone открой именно так:
   `https://kicker220203-dot.github.io/friday-v0.1/index.html?v=023dev`

## Если снова старая версия

1. Удали иконку Пятницы с рабочего стола iPhone.
2. Открой ссылку с `?v=023dev` в Safari.
3. Нажми обновить страницу 2–3 раза.
4. Потом добавь на экран Домой заново.
