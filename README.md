# Пятница v0.5-web PWA

Максимальная веб-версия без внешнего AI.

## Есть сейчас

- глаза, моргание, idle-анимации;
- касания, двойной тап, долгое нажатие;
- голосовые ответы;
- голосовое управление;
- голосовая сессия после двойного тапа;
- таймеры;
- простая память через localStorage;
- команды «запомни», «что ты помнишь», «забудь»;
- режимы: обычный, рабочий, тихий, сон;
- Вторник Lite без доступа к личным данным;
- меню памяти и таймеров;
- тест голоса, тест микрофона, сброс зависания;
- автоочистка старых service worker/cache.

## Нет сейчас

- внешнего AI;
- постоянного wake-word «Пятница»;
- Face ID / Owner Mode;
- настоящих системных уведомлений iOS;
- фоновой работы.

AI не добавлен специально: API-ключ нельзя хранить прямо в сайте.

## Как обновить GitHub Pages

1. Распакуй архив.
2. Загрузи содержимое папки `friday_v0_5_web_pwa` в корень репозитория.
3. Commit: `Update to v0.5-web`
4. Открой:
   `https://kicker220203-dot.github.io/friday-v0.1/index.html?v=05web`
