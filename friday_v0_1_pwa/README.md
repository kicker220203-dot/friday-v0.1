# Пятница v0.1 PWA

Это первый прототип Пятницы без AI и без голоса.

## Что уже есть

- чёрный фон;
- голубые глаза;
- моргание;
- idle-анимации;
- реакция на тап;
- двойной тап — внимание;
- долгое нажатие — меню;
- сон / пробуждение;
- тихий режим;
- preview режима Вторника;
- базовая OLED-защита через микродвижения и sleep.

## Как запустить на компьютере

Открой `index.html` в браузере.

Для полноценной PWA-установки лучше запускать через локальный сервер:

```bash
python -m http.server 8000
```

Потом открыть:

```text
http://localhost:8000
```

## Как поставить на iPhone как приложение

Нужно залить папку на HTTPS-хостинг:

- GitHub Pages;
- Netlify;
- Vercel;
- любой статический HTTPS-хостинг.

Потом на iPhone:

1. Открыть ссылку в Safari.
2. Нажать «Поделиться».
3. Выбрать «На экран Домой».
4. Запустить с иконки «Пятница».

## Управление

- один тап — лёгкая реакция;
- двойной тап — внимание;
- тап по глазу — забавная реакция;
- много тычков — раздражение / игнор;
- долгое нажатие — меню;
- клавиша M на компьютере — меню;
- клавиша S — сон;
- клавиша W — пробуждение.

## Важно

Это v0.1. Здесь специально нет AI, голоса и памяти.
Цель — проверить, ощущаются ли глаза и поведение живыми.
